# Resumo: Matemática para Casos Reais
Conceito    Uso real

União   -   juntar permissões, tags, dados de múltiplas fontes

Interseção -   encontrar elementos em comum entre listas, usuários, permissões

Pertinência -   validar se um item existe em um conjunto de valores permitidos

Subconjunto -   verificar se requisitos mínimos foram atendidos

Conjunto vazio  -   identificar ausência de dados ou resultados

Fatorial    -   calcular quantas ordens ou arranjos são possíveis

# Conclusão

Esses conceitos são fundamentais não apenas em matemática, mas também em software real. Eles aparecem em:

APIs
autenticação e autorização
validação de dados
cruzamento de informações
consultas a banco de dados
algoritmos de contagem.
